# E. P. E. – Eddie’s Print Engine

High-quality 26-page personalized coloring & mini-quest books generated from user images.

## 🚀 Live Application
https://eddies-print-engine.streamlit.app

## 🧠 Architecture

E.P.E. follows a hybrid architecture:

- Public interface & documentation
- Private core rendering engine
- Secure payment & rate-limit backend
- Streamlit-based deployment

This repository contains:

- Public documentation
- Architecture overview
- Usage explanation
- Development roadmap

The rendering engine and monetization backend remain private.

---

## ✨ What It Does

Users upload images.

The engine generates:

- 1 Intro page
- 24 quest pages
- 1 QR outro page
- Print-ready PDF (300 DPI, KDP-safe)

Modes:
- Kids Mode (XP, shapes, movement)
- Senior Mode (calm observation)
- Pre-Reader Mode
